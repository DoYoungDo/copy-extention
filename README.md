1. 右键复制绝对路径
> 会将选择的文件或目录的绝对路径复制到剪切板，在需要使用的地方直接粘贴即可

2. 右键复制目录结构
> 会将目录结构默认以下方的文本形式复制到剪切板，在需要使用的地方直接粘贴即可

```
└─testext
    ├─.hbuilderx
    │    └─launch.json
    ├─README.md
    ├─extension.js
    └─package.json
```

3. 在设置->插件设置->复制目录结构中可以进行一些相关设置
